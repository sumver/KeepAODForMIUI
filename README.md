# KeepAODForMIUI
让MIUI的万象息屏永不熄灭（不会因为静置、环境黑暗而关闭）


使用方法：<br>
1、安装apk文件<br>
2、在xposed框架比如LSPosed内勾选万象息屏<br>
3、强制停止万象息屏APP后立即生效<br>
